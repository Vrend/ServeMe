# ServeMe
A simple Web server

## Features

This server supports both GET and HEAD requests, and
can deliver html, css, javascript, images, binary data, etc.

## Usage

```
1. Clone the repository
2. Run 'make'
3. Add your files to the 'files' directory
4. Run the server executable
```
